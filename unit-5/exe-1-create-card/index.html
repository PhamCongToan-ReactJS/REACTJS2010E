Flex

This is shorthand for flex-grow, flex-shrink and flex basis combined. The second and thrid parameter (flex-shrink and flex-basis) are optional. Default is 0 1 auto.

Flex Grow

This defines the ability for a flex item to grow if necessary. It accepts a uitless value that serves as a proportion. It dictates what amount of the available space inside the flex container the item should take up.

Flex Shrink

This defines the ability for a flex item to shrink if necessary. Negative numbers are invalid.
